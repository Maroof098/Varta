# 💬 Varta – Modern Real-Time Chat Platform

Varta is a modern full-stack real-time messaging platform built with React, Firebase, and Vite. It provides a fast, secure, and elegant communication experience with a premium UI inspired by modern chat applications.

🔗 https://your-vercel-url.vercel.app

- 🔐 Firebase Authentication
- 💬 Real-time Chat
- 👥 User Discovery
- 🌙 Dark / Light Theme

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Maroof098/Varta.git
```

Go into the project folder:

```bash
cd Varta
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## 🔥 Firebase Configuration

Create a `.env` file and add:

```env
VITE_FIREBASE_API_KEY=YOUR_API_KEY
VITE_FIREBASE_AUTH_DOMAIN=YOUR_AUTH_DOMAIN
VITE_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
VITE_FIREBASE_STORAGE_BUCKET=YOUR_STORAGE_BUCKET
VITE_FIREBASE_MESSAGING_SENDER_ID=YOUR_SENDER_ID
VITE_FIREBASE_APP_ID=YOUR_APP_ID
VITE_FIREBASE_MEASUREMENT_ID=YOUR_MEASUREMENT_ID
```
